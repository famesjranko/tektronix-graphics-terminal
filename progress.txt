# Ralph Progress Log
Started: 2026-03-07
Project: Tektronix Graphics Terminal

## Codebase Patterns
- Use ES6 modules with `<script type="module">` in HTML
- Import colors from `./utils/colors.js` - exports COLORS object and DEFAULT_COLOR
- CSS variables match JS constants: --tek-primary = COLORS.primary
- TekCanvas: getPersistentContext() for final drawings, getAnimationContext() for previews
- Coordinates: toNormalized(x,y) and fromNormalized(x,y) convert between pixels and 0-1 range
- TekCanvas exports getWidth()/getHeight() for sizing calculations
- VectorRenderer: pass context + normalized coords, handles pixel conversion internally
- VectorRenderer uses _crisp() for sharp 1px lines (Math.floor(coord) + 0.5)
- PlotAnimator: use enqueue(command) to add commands, extends EventTarget for 'complete' event
- PlotAnimator takes TekCanvas and VectorRenderer in constructor
- ToolManager: takes TekCanvas, VectorRenderer, and PlotAnimator in constructor
- ToolManager: use addCommand(cmd) to add to history and queue animation, undo()/redo() for history
- Tools: extend BaseTool, use setCommandCallback(fn) to emit commands to ToolManager
- Mouse events: attach to animationCanvas, convert coords with toNormalized(), pass to active tool
- Fonts: import from ./fonts/hershey.js - HERSHEY_FONT object, getCharData(), textToCommands()

## 2026-03-07 - US-012
- What was implemented: Hershey Simplex vector font data for text rendering
- Files changed:
  - js/fonts/hershey.js - new file with stroke-based font data
- **Learnings for future iterations:**
  - HERSHEY_FONT object maps characters to {width, strokes} where strokes are [[x1,y1,x2,y2], ...]
  - Coordinates normalized with character height = 1, widths vary per character (typically 0.3-0.9)
  - Includes A-Z, 0-9, punctuation, and many additional symbols
  - Lowercase letters map to uppercase (same glyphs)
  - Helper functions: getCharData(char), measureText(text, spacing), textToCommands(text, x, y, scale, spacing)
  - textToCommands() returns array of line commands ready for PlotAnimator
  - baseHeight is 0.03 in normalized coords, scale parameter multiplies this
  - Unknown characters render as placeholder box
